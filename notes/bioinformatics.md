# fastq 

## Downsample fastq

### BBMap's reformat.sh

* Downsampling using total number of bases. 

    ```
    reformat.sh in1={input.illumina_r1} \\
                in2={input.illumina_r2} \\
                out1={params.unzipped_r1} \\
                out2={params.unzipped_r2} \\
                samplebasestarget=$BASECOUNT \\
                sampleseed={params.seed_random}
    ```
